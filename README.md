# ✈️ AI Travel Planner (MERN Stack)

---

## 🧠 Project Overview

The **AI Travel Planner** is a full-stack web application built using the MERN stack that helps users plan, organize, and manage their travel itineraries efficiently. It provides an interactive platform for creating trips, managing travel details, and receiving AI-assisted suggestions for better travel planning.

The system also includes authentication and user management for a personalized experience.

---

## 🎯 Objective

- Simplify travel planning for users  
- Provide structured trip management  
- Enable personalized travel organization  
- Improve user experience with AI-assisted suggestions
